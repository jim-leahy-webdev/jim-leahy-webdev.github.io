Week 2

In week 2 of this course, we learned a couple of notable things:

  • CSS Preprocessing
  • Branching & Merging
  • Creating a viewable web page in Github

Specifically, our course materials focused on the use of preprocessors like Sass and SCSS, allowing the use of more flexible, advanced features inside our stylesheets. By scripting in variables and functions, and nesting various style elements, one can create readable, functional CSS that is easier to maintain and edit.

While we expanded our understanding of the tools used to improve our CSS, we also spent time diving into the use of branching and pull requests. By creating a branch and a pull request, we can stage a change for review. This is a useful process when working as part of a team with an editor or on a software development team, where multiple people are involved in the process of coding and reviewing.
