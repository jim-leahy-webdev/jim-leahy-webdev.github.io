CSS Grid gives developers control over the layout of content. Instead of using hacks to try to fit content into specific areas of a page, CSS Grid controls the flow and allows you to set a target for where a grid box appears.

CSS Grid can automatically place and position grid items depending on the size of the screen. In the example above, the arrangement of the grid items is fixed, regardless of how wide the screen is.
