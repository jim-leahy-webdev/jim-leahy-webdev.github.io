Carousel

This image carousel is built using Bootstrap components. Bootstrap is designed specifically to be mobile-friendly and is generally optimized for responsive content display. Even so, there are times where Bootstrap components need adjustment. This carousel is no different.

The carousel images are my own. I had as much fun making the pizzas as I did creating this carousel!

When designing page contents, it's critical to view and test your pages on a variety of screen sizes or browser viewports. In early testing of this carousel, the text content on the images would disappear at smaller window sizes. The carousel here is a default bootstrap component and required some adjustment to allow for visibility on smaller viewports. As you resize this window on a desktop, you can see some of the breakpoints, and at smaller widths, the text would disappear. Adjusting some styles and classes for these carousel items preserved text visibility. I also reduced the text size on smaller viewports, allowing it fit better on smaller screens.
